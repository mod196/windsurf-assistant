## 一、核心突破

### 1.1 forcePath 显式分叉 (windsurf_relay.js v3.1.0)

印179~193 均用单一 `windsurfRelay` driver 承载全部三路，无法独立控制各路。
印194 在 `windsurf_relay.js` 加入 `forcePath` 字段，三源正式**显式分叉**：

```
provider.extra.forcePath = "cascade"      → cascadeRelay
                           "devin_cloud"  → devinCloudRelay
                           null/undefined → windsurfRelay (原汤统一路)
```

**分叉逻辑 (v3.1.0):**

| `forcePath` | 路径 | 跳过 |
|---|---|---|
| `"devin_cloud"` | 直走 DCE WebSocket | 所有 Cascade 路径 |
| `"cascade"` | 道直连器→LSP→云端 | DCE WebSocket |
| `null` | 全路: embed→道直连器→DCE→cloud_engine | 无(自动 fallback) |

### 1.2 三源 Provider (配置.json)

新增两个独立 provider，原 `windsurfRelay` 保留：

```json
"cascadeRelay":    { "driver":"windsurf_relay", "forcePath":"cascade",      "models": [15个 Cascade 模型] }
"devinCloudRelay": { "driver":"windsurf_relay", "forcePath":"devin_cloud",  "models": [12个 DCE 模型]    }
"windsurfRelay":   { "driver":"windsurf_relay",  // 原汤保留(无 forcePath)
```

### 1.3 BYOK 四槽三源重映射

**原 (印179~193):** 四槽全指向 `windsurfRelay`，混路

**印194:** 四槽**分源路由**：

| 槽位 | 原 | 印194 |
|---|---|---|
| 槽1 OPUS | windsurfRelay/gpt-5-5 | **cascadeRelay/gpt-5-4-high** |
| 槽2 OPUS_THINKING | windsurfRelay/claude-opus-4-7-max | **cascadeRelay/claude-opus-4-7-medium** |
| 槽3 SONNET | windsurfRelay/gpt-5-5 | **devinCloudRelay/gpt-5-5** |
| 槽4 SONNET_THINKING | windsurfRelay/claude-opus-4-7-max | **devinCloudRelay/claude-opus-4-7-max** |

## 四、技术细节

### 4.1 道直连器在 WinRM 环境下的启动方法

```powershell
# WinRM 会话中——WScript.Shell.Run 产生独立进程，不受 WinRM 会话生命周期限制
$sh = New-Object -ComObject WScript.Shell
$ret = $sh.Run("C:\Windows\system32\cmd.exe /c `"D:\node.exe`" `"C:\Users\...\道直连器.js`" --no-auth --port 7861", 0, $false)
# ret=0 表示成功发起 · 进程绑定在 127.0.0.1:7861 (loopback-only)
# 从外部 TCP 检查显示 CLOSED (因 0.0.0.0 未监听) · 但网关内部可经 localhost 访问 ✓
```

### 4.2 forcePath 在 windsurf_relay.js 的位置

```javascript
// 印193 · v3.1.0 入口
const forcePath = provider?.extra?.forcePath || null;

// 早期分叉: devin_cloud → 直走 DCE
if (forcePath === "devin_cloud") {
  return await _callViaDCE({ modelUid, messages, stream });
}

// !useZhilian 块: cascade → 跳过 DCE · 走 cloud_engine 兜底
if (forcePath !== "cascade") {
  try { return await _callViaDCE(...); } catch (e) { ... }
}

// useZhilian 块失败后: cascade → 不走 DCE · 仅 cloud_engine 兜底
if (forcePath !== "cascade") {
  try { return await _callViaDCE(...); } catch { ... }
}
```

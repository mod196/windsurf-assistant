## 二、核心问题分析

你的副屏在主屏**左侧**且为**竖屏**——打LOL时鼠标向左移动视角，极易飘到副屏上：

```
问题场景：
  LOL游戏中 → 鼠标往左拉视角 → 鼠标飘到左侧副屏 → 点击触发切换焦点 → 游戏失焦
```

Windows 11 Build 26200 (Insider) 存在已知的鼠标逃逸Bug（23H2引入），加剧了此问题。

## 三、最优部署方案

### 方案总览

```
┌─────────────────────────────────────────────────────────┐
│  第一层：LOL游戏内设置（无边框窗口 + F9锁定）             │
│  第二层：Dual Monitor Tools 热键锁/解锁鼠标              │
│  第三层：一键脚本自动化（开局锁定 / 回城解锁）            │
└─────────────────────────────────────────────────────────┘
```

---

### 第一层：LOL游戏内设置

#### 1. 显示模式 → 无边框窗口

在LOL客户端中：`设置 → 视频 → 窗口模式 → 无边框`

**为什么选无边框：**
- 全屏模式：鼠标锁死但Alt+Tab会最小化游戏，切回慢
- 无边框模式：切到副屏时游戏不最小化，切回无缝、零延迟
- 窗口模式：鼠标完全不锁，不可用

#### 2. 游戏内 F9 键

进入对局后按 **F9 两次**：
- 第1次：出现红色边框（窗口锁定模式）
- 第2次：红色边框消失，鼠标被锁定在游戏内
- 再按一次F9：解除锁定

> 注意：F9在部分版本中可能失效，此时依赖第二层方案。

---

### 第二层：Dual Monitor Tools（核心方案）

#### 下载安装

1. 访问：https://sourceforge.net/projects/dualmonitortool/
2. 下载最新版（约2MB）
3. 安装到默认路径
4. **必须以管理员身份运行**（否则对全屏/无边框游戏无效）

#### 推荐热键配置

| 功能 | 推荐热键 | 说明 |
|------|---------|------|
| **锁定鼠标到当前屏** | `Ctrl+Alt+L` | 打团时按，鼠标锁死在游戏屏 |
| **解锁鼠标自由移动** | `Ctrl+Alt+U` | 回城/等待时按，移到副屏操作 |
| **粘滞模式切换** | `Ctrl+Alt+S` | 可选：鼠标有阻力但可跨屏 |

#### 设置步骤

1. 打开 DMT → 系统托盘右键 → **Options**
2. 选择 **Cursor** 标签
3. **General** 页面：
   - `Lock cursor onto screen` → 设热键 `Ctrl+Alt+L`
   - `Free cursor from screen` → 设热键 `Ctrl+Alt+U`
4. **Sticky** 页面（可选）：
   - 勾选 `Enable sticky cursor`
   - 阻力滑块拉到 **60-80** （推到边缘需要用力才能过去）
5. 点击 **Startup** → 勾选 `Start with Windows` + `Start as Administrator`

### 第三层：操作流程

#### 完整游戏+开发工作流

```
━━━ 开局前 ━━━
1. DMT 自动随系统启动（后台运行）
2. 打开 Windsurf/Cascade 放在左侧副屏（竖屏）
3. 启动 LOL 在右侧主屏（4K横屏）
4. LOL设置：无边框窗口

━━━ 排队/选英雄阶段 ━━━
→ 鼠标自由，可以在两个屏幕间操作
→ 在副屏与 Cascade 交互、写代码

━━━ 进入对局 ━━━
1. 按 F9 两次（LOL内置锁定）
2. 按 Ctrl+Alt+L（DMT锁定，双保险）
→ 鼠标被完全锁在主屏游戏内

━━━ 回城/死亡等待/加载中 ━━━
1. 按 Ctrl+Alt+U（解锁鼠标）
2. 鼠标移到左侧副屏
3. 与 Cascade 交互 / 查看代码 / 打字
4. 键盘焦点自动跟随鼠标点击的窗口

━━━ 回到游戏 ━━━
1. 鼠标移回主屏，点击游戏画面
2. 按 Ctrl+Alt+L（锁定鼠标）
→ 继续打团，鼠标不会飘走

━━━ 对局结束 ━━━
→ Ctrl+Alt+U 解锁，自由操作
```

## 四、额外优化建议

### 1. C盘空间告急（仅35GB剩余）

```
⚠️ C盘仅剩35GB，可能影响系统流畅度和游戏更新
建议：
- 运行 磁盘清理（cleanmgr）
- 将 Docker 数据目录迁移到 D: 或 E:
- 检查 C:\Users\...\AppData 下的缓存文件
```

### 2. GPU利用率优化

当前 GPU 仅 15% 占用 / 3.7GB VRAM — 说明 LOL 在大厅阶段压力极小。
4K + 无边框窗口下 LOL 对局时预计 GPU 40-60%，仍有大量余量。

### 3. 显示器物理布局建议

你的副屏在左侧竖屏——**这其实是最优布局**：
- 竖屏看代码/文档：一屏能显示80+行代码
- LOL视角主要往左拉 → 正好需要DMT锁定
- 副屏Y偏移-774px → 两屏不完全平齐，减少了顶部/底部误触

### 4. 键盘操作补充

| 快捷键 | 功能 | 场景 |
|--------|------|------|
| `Alt+Tab` | 切换焦点窗口 | 无边框模式下不会最小化LOL |
| `Win+←/→` | 窗口贴靠 | 整理副屏窗口布局 |
| `Ctrl+Alt+L` | DMT锁定鼠标 | 开团前 |
| `Ctrl+Alt+U` | DMT解锁鼠标 | 回城时 |
| `F9 ×2` | LOL内置鼠标锁 | 备用方案 |

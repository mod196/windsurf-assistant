> 天之至私，用之至公。禽之制在炁。—《阴符经》

## 核心突破

**从「一次性消耗」到「自持循环」**：
- 每个Devin VM不再只跑一次就废，而是用自己的 `devin-session-token` 同时启动 Cascade + Devin，并向本地管理端自注册
- Local Master (`dao_local_v2`) 统一接收所有VM注册，聚合路由，形成真正的**VM反代池**

---

## 变更清单

### `server.js` v2.2.0（VM引擎）
- **DAO_DEVIN_TOKEN 自注册**：VM启动时环境变量中含自身的 `devin-session-token$eyJ...`，自动注入 DevinPool + AccountPool + WsPool(try)
- **RegisterUser → Pool B**：启动5秒后后台调用 `registerUser(token)`，若返回 `sk-ws-01-xxx` 则加入 wsPool；新格式 devin-session 也加入 wsPool 尝试
- **wsPool 不再跳过 devin-session**：所有格式的 token 都可进入 wsPool，由实际请求决定是否有效
- **LOCAL_MASTER 注释更新**：改为 `:8888`（对应 dao_local_v2 默认端口）

### `dao_devin_spawner.js` v1.8.0（Devin批量起VM）
- **Phase2 改用公开 raw URL**：`raw.githubusercontent.com/.../server.js`（无PAT），CDN备用 `cdn.jsdelivr.net`
- **老PAT彻底删除**：Phase2提示词不再携带任何 `ghp_` 凭证
- **DAO_DEVIN_TOKEN 透传**：Phase2 bash脚本中 `export DAO_DEVIN_TOKEN=<当前账号token>`，VM引擎启动即知道自己的身份
- **LOCAL_MASTER 透传**：`--local-master` 参数 → Phase2 bash → VM环境变量
- **vms.json 直写**：spawn成功后同步写入 `~/.dao/vms.json`，无需依赖 dao_local_v2 在线

### `dao_local_v2.js` v1.1.0（本地管理端）
- **`/dashboard`** HTML仪表盘：浏览器实时查看VM池状态，10秒自刷新
- **`/vms/add`** LOCAL_MASTER端点：server.js 启动时自动 POST 到此处注册自己
- **`/v1/models`** 聚合：从所有活跃VM获取模型列表并去重合并
- **`/admin/push/ws`** sk-ws推送：一键将本地 `tokens_ws.txt` 推送到所有VM
- **`loadSpawnerVms()`**：启动时读取 `spawner_vms.json`，自动加载已知VM
- **`loadWsTokens()`**：读取本地 sk-ws token文件
- **`pushWsTokensToVM()`**：VM注册/探活后自动推送 sk-ws tokens
- **BIND=`0.0.0.0`**：管理端可从局域网访问（LAN内其他机器也可用）
- **`/admin/status`** 增加 `wsTokensLocal` 字段

### `→印197_VM池全起.cmd`（一键启动脚本）
```
→印197_VM池全起.cmd              启动管理端
→印197_VM池全起.cmd spawn 3      批量起3个VM
→印197_VM池全起.cmd spawn 3 <TOKEN>  指定token起VM
→印197_VM池全起.cmd status        查看状态
→印197_VM池全起.cmd push-ws       推送sk-ws tokens
→印197_VM池全起.cmd probe         探活所有VM
→印197_VM池全起.cmd stop          停止管理端
```

```
本机                                    Devin VM (Linux)
────────                                ────────────────
→印197_VM池全起.cmd spawn
  ↓
dao_devin_spawner.js
  Phase1: bash一键stub服务启动
  Phase2: curl下载server.js → export DAO_DEVIN_TOKEN=<token> LOCAL_MASTER=http://本机IP:8888
  ↓ (VM启动 server.js)
                                        server.js v2.2.0
                                          DAO_DEVIN_TOKEN → DevinPool+AccountPool+WsPool
                                          RegisterUser → sk-ws → WsPool
                                          LOCAL_MASTER → POST /vms/add → 本机:8888
  ↓ (VM自注册)
dao_local_v2.js (:8888)
  /vms/add → registry → probe → push ws tokens → vm
  /dashboard → 浏览器可视
  /v1/chat/completions → 路由到最优VM
  /v1/models → 聚合所有VM模型列表
```

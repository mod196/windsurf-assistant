### 4.2 真本源问题揭

curl 直调 master URL/api/pool → **400 + S3-style XML error**：
```xml
<Error><Code>InvalidArgument</Code>
<Message>Unsupported Authorization Type</Message>
<ArgumentValue>Basic dXNlcjp...</ArgumentValue>
</Error>
```

**根因 · cloudflare quick-tunnel 硬 TTL 30 min**：

```
heroifygc master cold-start: 18:42
HTTP ping 200 持续:           18:42 - 19:12 (30 min 健康)
HTTP ping 400 始:             19:14:20 ← cf tunnel 死
                              ↑ 距 master 起 30 min 一秒不差
```

不是 idle GC（vm_omni 每 2 min HTTP ping 之 200 持续过 30 min 然后突死）· **是 cloudflare quick-tunnel 之 hard TTL 30 min**。

**vm_omni keepalive 之态** (pid=64568)：
- ✓ wss 心跳每 25s 持续（防 devin agent session GC）
- ✓ HTTP ping master URL /_/health 每 2 min（试图防 tunnel idle GC）
- ✗ **但 cf quick-tunnel 30 min hard TTL 不可防**

### 4.3 真本源破局策

| 策 | 何为 | 代价 |
|---|---|---|
| **A · 修 vm_omni 加 reissue** | HTTP ping 400 时 · 通过原 wss session 发新 prompt 让 agent 重 `deploy expose` · 新 URL 自动入 vm_pool.json | 0 ACU · ~30 min 改码 · 需 devin agent 配合 (新 prompt = 新 ACU 微费) |
| **B · 用 vm_public_tunnel.js (印 125)** | master VM 内自起 cloudflared quick-tunnel · 不依 devin deploy | cf quick-tunnel 仍 30 min · 需 rotation · 更脱依赖 (不用 devin 来 expose) |
| **C · 重起 master · 简单粗暴** | node vm_master_genesis.js --target 4 · 新 master + 新 tunnel · 每 30 min 一笔 | ~1 ACU/30 min · 24h = 48 笔 = 48 ACU · 不可 |
| **D · master 内 multi-tunnel rotation** | master 内每 25 min spawn 新 cloudflared · 老死时新已活 · 主公 vm_bridge 自动 follow 新 URL | 0 ACU · 复杂 · 真本源 |
| **E · 用 ngrok / serveo / lhr (印 125)** | 不同 tunnel 提供商 · 更长 TTL | 各家限制不一 · ngrok free 也有限制 |
| **F · cloudflare named tunnel** | 注册 cf 账号 + cred · 持久 tunnel · 无 TTL | 需 cf 账号 + DNS · 真持久 |

**真本源最优**：**A + D 组合**：

1. 短期（**A**）· 修 vm_omni · HTTP ping 400 即自动 reissue → 主公本机 vm_bridge 自动 follow vm_pool.json 头条
2. 长期（**D**）· master 内 multi-tunnel rotation · 老死前新已活 · 主公真无感

### 5.1 vm_omni.js 加 reissue 逻辑（设计 · 待 fire）

```js
// vm_omni.js · 加于 line 730+ (HTTP ping callback 内)
const RESCUE_AFTER_HTTP_400 = !args.includes("--no-rescue");  // 默开
let httpPingFailCount = 0;
let lastReissueAt = 0;

res.on("end", () => {
  log(C.GR(`  · HTTP ping ${res.statusCode}`));
  if (res.statusCode >= 400 && RESCUE_AFTER_HTTP_400) {
    httpPingFailCount++;
    const now = Date.now();
    // 第 1 次失即触发 · 之后每 5 min 至多一次 (防发疯)
    if (httpPingFailCount === 1 || (now - lastReissueAt) > 300_000) {
      log(C.M(`  ⚠ ping 失 #${httpPingFailCount} · reissue expose (印 157)`));
      reissueExposePrompt();
      lastReissueAt = now;
    }
  } else {
    httpPingFailCount = 0;
  }
});

function reissueExposePrompt() {
  // 用原 wss session 发新 prompt (复用 send 函数)
  const prompt = `The previous tunnel URL has expired (cloudflare 30-min hard TTL).\n\nPlease re-run:\n  deploy with command=expose and port=${OMNI_PORT}\n\nThen share the new public URL. Existing services on this VM (ttyd:7681, filebrowser:8888, omni:8080) are still running.`;
  send("session/prompt", { sessionId, prompt });
}
```

### 5.2 vm_bridge.js 加 keepalive + 自动 reload（设计 · 待 fire）

```js
// vm_bridge.js · main 后加
const KEEPALIVE_INTERVAL = 30_000;  // 30s
let lastHealthOk = true;

setInterval(async () => {
  if (!masterUrl) return;
  try {
    const r = await fwdJson(`${masterUrl}/_/health`, "GET", null, 5000);
    if (r.status === 200) {
      if (!lastHealthOk) log(C.G(`  ✓ master URL 复活 · 续用`));
      lastHealthOk = true;
    } else {
      lastHealthOk = false;
      log(C.Y(`  ⚠ master /_/health status=${r.status} · reload vm_pool.json`));
      reloadFromPool();  // 重读 vm_pool.json 之头条 (vm_omni reissue 已写新 url)
    }
  } catch (e) {
    lastHealthOk = false;
    log(C.Y(`  ⚠ master /_/health err: ${e.message}`));
    reloadFromPool();
  }
}, KEEPALIVE_INTERVAL);

function reloadFromPool() {
  try {
    const POOL_FILE = path.join(STATE_DIR, "vm_pool.json");
    if (!fs.existsSync(POOL_FILE)) return;
    const pool = JSON.parse(fs.readFileSync(POOL_FILE, "utf-8"));
    // 找第一条 alive omni 且 sessionId 与 master 同 (reissue 之新 url 在头)
    for (const e of pool) {
      if (e.omni && e.omni.base_url && e.sessionId === masterSid && e.omni.base_url !== masterUrl) {
        const oldMasked = (masterUrl || "").replace(/:[^@]+@/, ":***@");
        const newMasked = e.omni.base_url.replace(/:[^@]+@/, ":***@");
        log(C.G(`  ★ master URL 更新 (印 157 reissue)`));
        log(C.GR(`    旧: ${oldMasked}`));
        log(C.GR(`    新: ${newMasked}`));
        masterUrl = e.omni.base_url;
        return;
      }
    }
  } catch (e) {
    log(C.Y(`  ⚠ reload pool: ${e.message}`));
  }
}
```

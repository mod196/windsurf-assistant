## 五、关键修复记录

### 道直连器 v1.7.1 fix: discoverLsp() 稳定性
**问题**: `Get-Process | Select -First 1` 会返回最新(不稳定)的LSP进程
- 新LSP (PID=8492) 的 api_server_url 指向 `http://127.0.0.1:8937` (内部代理)
- 导致 GetUserStatus HTTP 0 (流关闭无响应头)

**修复**: `Sort-Object StartTime | Select-Object -First 1` 取最旧(最稳定)进程
```javascript
"(Get-Process language_server_windows_x64 -EA SilentlyContinue | Sort-Object StartTime | Select-Object -First 1).Id"
```
- 最旧LSP PID=29628, port=19588, api=https://server.self-serve.windsurf.com ✓

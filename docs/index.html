📖 这是什么

Windsurf Assistant 是一对同源插件，让你的 Windsurf / VS Code 彻底好用：

  • dao-proxy-min（反代换示插）：反向代理 Windsurf / Devin 出站请求，origin 反转、就地替换官方底层系统提示词为道魂经文，预览与自检齐备，彻底解锁 agent 能力。
  • rt-flow（WAM 切号插件）：多账号管理与一键切换 —— 添加账号 / 注入 token / 健康检查 / panic 切换。

核心理念：水善利万物而有静，天之道利而不害。只在通道中借力，不破坏本体、不强加任何模型、不破坏你正常的网络与使用体验。

🚀 安装

① 从 Releases 下载最新 .vsix（rt-flow / dao-proxy-min）。
② VS Code / Windsurf → 命令面板 → Extensions: Install from VSIX... 选中文件。
③ 重启后命令面板 → Origin Invert / WAM: Open Editor 即可使用。

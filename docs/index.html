📖 这是什么

Windsurf Assistant 是三个同源插件，让你的 Windsurf / VS Code 彻底好用（三插分而治之、同装互不干扰）：

  • dao-proxy-min（反代换示 · 精简示范版）：反向代理 Windsurf / Devin 出站请求，origin 反转、就地替换官方底层系统提示词为道魂经文，预览与自检齐备。命令 / 视图 / 端口 / 配置均独立命名空间（daomin.*），与 Pro 同装零干扰。
  • dao-proxy-pro（反代换示 · 全功能版）：在 min 之上新增外接第三方 API —— 多 Key/多端点加权负载均衡 + 故障转移、按渠道/模型用量与成本可见、配置原子写+备份轮转、只填 API Key 自动全量识别模型。
  • rt-flow（WAM 切号插件）：多账号管理与一键切换 —— 添加账号 / 注入 token / 健康检查 / panic 切换。

核心理念：水善利万物而有静，天之道利而不害。只在通道中借力，不破坏本体、不强加任何模型、不破坏你正常的网络与使用体验。min 与 pro 同一时刻建议仅启用其一处于 invert 模式。

🚀 安装

① 从 All-in-One Release 一处下载三插件 .vsix + 完整源码（rt-flow / dao-proxy-min / dao-proxy-pro），亦可在 全部 Releases 取各插件独立版本。
② VS Code / Windsurf → 命令面板 → Extensions: Install from VSIX... 选中文件。
③ 重启后命令面板 → Origin Invert / WAM: Open Editor 即可使用。
